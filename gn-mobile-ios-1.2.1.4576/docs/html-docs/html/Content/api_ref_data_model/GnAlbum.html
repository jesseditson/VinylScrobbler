Property Type Description
GnAlbum Object Album matches. Collection of audio recordings. Provides access to album cover art when received from a query object with content enabled in lookup data.
artist GnArtist Album's artist
contributor GnContributor A Contributor refers to any person who plays a role in a Work. Actors, Directors, Producers, Narrators, and Crew are all considered a Contributor. Popular recurring Characters such as Batman, Harry Potter, or Spider-man are also considered Contributors
artistType String Contributor's artist type.
biography GnContent Contributor's biography content object. See GnAlbum > content for object properties.
biographyVideo String Contributor's biography when received from a video response.
birthDate String The contributor's birth date in UTC format.
birthPlace String The contributor's birth place.
collaborator GnContributor Contributor that collaborated with this contributor in the context of the returned result. See GnAlbum > artist > contributor for object properties.
collaboratorResult Boolean Flag indicating if this is a collaborator result
contents GnContent
Enumerator		 Content objects. See GnAlbum > content for object properties.
deathDate String The contributor's date of death in UTC format.
deathPlace String The place where the contributor died.

externalIds GnExternalId

genre(GnDataLevel level) String Contributor's genre
gnId String Gracenote identifier
gnUId String Contributor's Gracenote unique Id
image GnContent Contributor's image content object. See GnAlbum > content for object properties.
isFullResult Boolean Indicates if this is a partial or full metadata result.
