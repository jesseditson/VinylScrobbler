Embedded Gracenote Database (GDB)

To enable offline functionality in devices that are not connected to the Internet, Gracenote provides a distilled and compressed embedded Gracenote Database (GDB) that you can embed inside devices. The embedded GDB is used by MusicID (CD, Text), PlaylistA set of tracks from a user’s music collection, generated according to the criteria and limits defined by a playlist generator., Cover Art, Artist Images and Link products.

The ideal scenario utilizes an embedded GDB augmented with online lookup capabilities. This combines fast local recognition of popular content with the added depth of coverage and up-to-date metadata available in the online Gracenote Database.

The initial pre-loaded embedded database is loaded onto the device at the time of production and is referred to as the Base Database. Over the lifetime of the product the Base Database may be updated with information for both newly released music, and music that has had resurgence in popularity since the creation of the Base Database. These updates help maintain optimal recognition rates over the life of the product.
