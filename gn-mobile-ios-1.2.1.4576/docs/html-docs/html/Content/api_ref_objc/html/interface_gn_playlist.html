Detailed Description

Experimental: Provides access to information regarding the underlying GNSDK playlist library. For working with user collections to create playlists see GnPlaylistCollection.

+ (nonnull NSString*) buildDate

Retrieves the Playlist SDK's build date string.

Returns
Note Build date string of the format: YYYY-MM-DD hh:mm UTC

Remarks:

You can call this API at any time after successfully getting a GnManager instance. The returned string is a constant. Do not attempt to modify or delete. Example build date string: 2008-02-12 00:41 UTC

Retrieves the Playlist SDK version string.

Returns
version string if successful

Remarks:

Example version string: 1.2.3.123 (Major.Minor.Improvement.Build) Major: New functionality Minor: New or changed features Improvement: Improvements and fixes Build: Internal build number
