- (nullable GnMusicIdFileInfo*) getByFilename: (nullable NSString *)  filename
error: (NSError **)  error 

Retrieves the GnMusicIdFileInfo object that matches the given file name.

Parameters
filename[in] File name or other string identifier of FileInfo to retrieve

Remarks:

Only GnMusicIdFileInfo objects that have file name information set through GnMusicIdFile::FileName can be retrieved with this function.

Note:

GnMusicIdFile does not enforce unique GnMusucIdFileInfo file names. Consequently, to use this function well, it is important to give unique file names to each GnMusicIdFileInfo added (such as the full path and file name). GnMusicIdFile does not check if the given file name given is a valid path or file name, nor is it required to be one. GnMusicIdFile only performs a match against the existing GnMusicIdFileInfo path and file names.
